# AlmondHab 🎧

Turn your hum into a beat.

- Hum → Beat generation
- Drum grid sequencer
- Real-time sound engine
- Export beats

## 🛠️ Tech
- HTML, CSS, JavaScript
- Web Audio API

## ▶️ How to Use
1. Click “Start Humming”
2. Tap or generate pattern
3. Play and export your beat
